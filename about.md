## Getting Started

To add your first blog post:
1. Create a new file in the `_posts` directory
2. Name it with the format: `YYYY-MM-DD-title.md`
3. Add front matter at the top (layout, title, date)
4. Write your content in Markdown
5. Commit and push to see it live!
